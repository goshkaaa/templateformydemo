# Структура базы данных

База данных: `conferences_rf`.

## ER-структура

```text
users 1 ─── n bookings n ─── 1 rooms
users 1 ─── n reviews  1 ─── 1 bookings
```

## users

Пользователи портала.

| Поле | Назначение |
| --- | --- |
| id | первичный ключ |
| login | уникальный логин, латинские буквы и цифры |
| password_hash | хеш пароля PHP `password_hash` |
| full_name | ФИО |
| phone | контактный телефон |
| email | e-mail |
| created_at | дата регистрации |

## rooms

Помещения для конференций.

| Поле | Назначение |
| --- | --- |
| id | первичный ключ |
| title | название помещения |
| type | `Аудитория`, `Коворкинг` или `Кинозал` |
| capacity | вместимость |
| description | описание |

## bookings

Заявки на бронирование.

| Поле | Назначение |
| --- | --- |
| id | первичный ключ |
| user_id | связь с пользователем |
| room_id | связь с помещением |
| event_date | дата начала конференции |
| payment_method | способ оплаты |
| status | `Новая`, `Мероприятие назначено`, `Мероприятие завершено` |
| created_at | дата создания заявки |

## reviews

Отзывы пользователей. Отзыв доступен только после того, как администратор изменит статус заявки.

| Поле | Назначение |
| --- | --- |
| id | первичный ключ |
| booking_id | связь с заявкой, уникальная |
| user_id | связь с пользователем |
| rating | оценка от 1 до 5 |
| text | текст отзыва |
| created_at | дата создания |
