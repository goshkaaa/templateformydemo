# Запуск проекта через OpenServer и phpMyAdmin

Проект сделан под обычный экзаменационный запуск на Windows: OpenServer, Apache/PHP, MySQL и phpMyAdmin.

## 1. Куда положить проект

Скопируйте папку проекта в каталог доменов OpenServer, например:

```text
OpenServer\domains\conferences-rf
```

Внутри должны лежать файлы:

```text
index.php
login.php
register.php
profile.php
booking.php
admin.php
config.php
sql\init.sql
```

## 2. Создание базы данных

1. Запустите OpenServer.
2. Откройте phpMyAdmin.
3. Перейдите во вкладку `Импорт`.
4. Выберите файл `sql/init.sql`.
5. Нажмите `Вперед`.

SQL сам создаст базу `conferences_rf`, таблицы и стартовые помещения.

## 3. Настройка подключения

По умолчанию в `config.php` указаны стандартные настройки OpenServer:

```php
const DB_HOST = '127.0.0.1';
const DB_NAME = 'conferences_rf';
const DB_USER = 'root';
const DB_PASS = '';
```

Если в вашей сборке OpenServer пароль MySQL отличается, измените только `DB_PASS`.

## 4. Адрес сайта

Откройте домен OpenServer в браузере:

```text
http://conferences-rf/
```

Если домен назван иначе, используйте имя своей папки в `domains`.

## 5. Что показать эксперту

- `index.php` - главная страница портала «Конференции.РФ».
- `register.php` - регистрация с проверкой логина, пароля, ФИО, телефона и e-mail.
- `login.php` - вход пользователя и администратора.
- `booking.php` - оформление заявки на бронирование.
- `profile.php` - личный кабинет, история заявок, слайдер и отзывы.
- `admin.php` - панель администратора с фильтрами, сортировкой, пагинацией и сменой статуса.

## 6. Данные администратора

```text
Логин: Admin26
Пароль: Demo20
```

Обычные пользователи создаются через страницу регистрации.

## 7. Типовая проверка

1. Зарегистрируйте пользователя.
2. Создайте заявку на странице `booking.php`.
3. Выйдите и войдите как администратор.
4. В `admin.php` смените статус заявки на `Мероприятие назначено`.
5. Войдите обратно пользователем.
6. В `profile.php` проверьте, что появилась возможность оставить отзыв.
