FAQ

Аккордеон для частых вопросов.

Замените тексты, изображения, категории в JSON и палитру в variables.css.
В демо-режиме данные сохраняются в localStorage. В полноценном режиме авторизация идет через Node.js API и MySQL.
Для MySQL нужен запуск через docker compose up --build. Простое открытие HTML не может подключиться к базе данных.
