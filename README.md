# Конференции.РФ

PHP/MySQL-проект для демонстрационного экзамена по варианту В2.

В проекте используется объектно-ориентированный слой для подключения к базе данных, авторизации и валидации. Bootstrap 5.3.8 хранится локально в `css/vendor`, поэтому интерфейс не требует доступа к интернету.

## Быстрый запуск в OpenServer

1. Скопируйте проект в папку доменов OpenServer.
2. Через phpMyAdmin импортируйте `sql/init.sql`.
3. Откройте `index.php` на домене проекта.

Настройки MySQL находятся в `config.php`. По умолчанию используется база `conferences_rf`, пользователь `root`, пустой пароль.

## Основные страницы

- `register.php` - регистрация пользователя.
- `login.php` - авторизация.
- `booking.php` - оформление заявки.
- `profile.php` - личный кабинет, слайдер и отзывы.
- `admin.php` - управление заявками.
